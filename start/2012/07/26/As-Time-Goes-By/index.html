小时间

时间总是不会让你轻易的看见它蜿蜒的维度，当你发现时，悠然间可能已葱葱几年。 弹指间的沧海桑田还有消融在记忆里的一个又一个，不被人注意，不被自己实现的小小的梦想或是期许。 恍然间，岁月仓惶，前行的脚步时而会显得有些许的彷徨趔趄，时而又显的笃定，才发现，尽管如此，此时的心，也已经可承载更多。
